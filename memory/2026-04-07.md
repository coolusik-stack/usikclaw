
## 브리핑 페이지 디자인 포맷 확정
- 사용자가 2026-04-07 브리핑 페이지 새 디자인을 최종 포맷으로 승인.
- 앞으로 아침 브리핑은 이 포맷을 기본값으로 사용.
- 특징:
  - 에디토리얼/매거진 스타일
  - 크림 종이 톤 배경
  - Playfair Display + DM Sans + DM Mono 타이포 조합
  - 상단 ticker
  - 좌측 메인 기사 / 우측 sticky sidebar
  - SVG 라인 일러스트 포함 (sun, chart, fox, chip, book, divider)
- 사용자는 아침 브리핑에 항상 '전날 새로 업데이트된 소식만' 넣기를 원함. 기존 발표/오래된 소식의 재인용은 제외.
