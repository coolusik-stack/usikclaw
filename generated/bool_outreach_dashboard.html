목표는 서울 핵심상권 전자담배점의 스마트콜/대표번호를 확보하고, BOOL 입점 제안을 위한 콜드 메시지를 테스트하는 것입니다. 지금 페이지는 강남역, 신논현/논현, 역삼/선릉 축의 1차 후보를 표로 정리하고, 메시지 타입과 우선순위를 바로 확인할 수 있게 만든 운영용 페이지입니다.

실행 순서

  1. A등급 12개 안팎에 먼저 메시지를 발송합니다.
  2. 타입 1, 2, 3 문안을 섞어 소량 테스트합니다.
  3. 응답 매장은 바로 전화 또는 방문으로 이어갑니다.
  4. 무응답 매장은 1~2일 뒤 1회만 후속 메시지를 보냅니다.
  5. 반응률이 확인되면 홍대/합정, 성수, 잠실로 확장합니다.

처음부터 100개를 한 번에 치기보다, 반응이 나오는 문안과 상권을 먼저 찾는 쪽이 효율이 높습니다.

평가 기준

  • 입지 점수: 핵심상권 메인 동선 여부
  • 전문성 점수: 전자담배 전문점 성격이 얼마나 분명한지
  • 접촉 가능성 점수: 전화번호, 지도 링크, 운영 흔적이 충분한지
  • A등급: 바로 메시지 + 전화 후보
  • B등급: 메시지 우선, 확인 후 진행

현재 표는 공개 웹 검색 기반 초안입니다. 실제 발송 전에는 운영 여부와 번호 정확성을 한 번 더 확인하는 전제가 맞습니다.

타입 2, 마진/실무형

안녕하세요. BOOL KOREA입니다. 매장에서 바로 판매 가능한 니코틴파우치 BOOL 제안드리려고 연락드렸습니다. 권장가 기준 매장 마진 구조가 괜찮고, 전자담배 전문점에서 제안드리기 좋은 상품입니다. 원하시면 간단한 소개자료와 조건 먼저 보내드리겠습니다.

타입 3, 짧은 테스트형

안녕하세요. BOOL KOREA입니다. 전자담배 매장용 니코틴파우치 입점 제안드리려고 연락드렸습니다. 부담 없이 보실 수 있는 짧은 소개자료 있는데 보내드려도 괜찮을까요?
