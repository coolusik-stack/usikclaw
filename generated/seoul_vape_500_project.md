# 서울 전자담배점 500개 확보 프로젝트

## 목표
- 서울 시내 전자담배 관련 매장 **500개 컨택포인트 확보**
- 최우선 기준은 **문자 수신 가능한 정확한 번호(010 / 0507)**
- 최종 목적은 BOOL 콜드아웃리치의 **실제 전환 가능한 풀(pool)** 구축

---

## 현재 기준점
- 현재 확보 매장: 266개
- 문자 가능한 번호: 261개
- 재확인 필요: 4개

즉, 남은 목표는 대략:
- 전체 매장 +234개
- 문자 가능한 번호도 최대한 동일 수준으로 확대

---

## 프로젝트 원칙

### 1. 숫자보다 정확도 우선
500개를 채우는 것이 목표지만,
핵심은 **문자 보낼 수 있는 실제 유효 번호**다.

### 2. 010 / 0507 우선
- 1순위: 010
- 2순위: 0507
- 3순위: 02 / 070 / 080 (보조 보관용)

### 3. 억지 추정 금지
번호가 애매하면 넣지 않거나 `재확인 필요`로 남긴다.

### 4. 지역 확장은 점진적으로
한 번에 서울 전체를 뒤지는 방식보다,
권역별 누적 수집이 정확도와 유지보수에 유리하다.

## 리스크

### 1. 중복
같은 매장이 다른 상호/주소 표현으로 중복될 수 있음
→ 주기적 중복 정리 필요

### 2. 번호 정확성
옛 번호, 고객센터 번호, 취급점 번호가 섞일 수 있음
→ 보수적으로 유지

### 3. 폐업/휴무
공개 웹 정보가 최신이 아닐 수 있음
→ 발송 전 최종 체크 필요

### 4. 서울 전역 커버리지 착시
숫자는 500개여도 품질이 낮으면 의미 없음
→ 항상 SMS 가능 번호 비율을 같이 추적

## claw 의견
500개는 충분히 갈 수 있다.
하지만 진짜 중요한 건 500이라는 숫자가 아니라,
**실제로 문자 보내도 되는 정확한 번호 풀을 크게 만드는 것**이다.

따라서 이 프로젝트는 앞으로도:
- 후보 확대
- 번호 정제
- 대시보드 갱신
을 반복하는 방식으로 운영하는 게 맞다.
