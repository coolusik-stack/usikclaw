How to read this

Quick operator guidance

  1. A moving last update time or fresh log lines means a worker is still alive.
  2. Running means the loop is active now. Completed means its output has already been folded into the dataset.
  3. If queue watcher keeps advancing but verification loop stays idle, the system is waiting on new candidates rather than being broken.

Tip: open this file directly or publish it as a static page. The companion JSON artifact can feed another frontend later without changing the source logs.
