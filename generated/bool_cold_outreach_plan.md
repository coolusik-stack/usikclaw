# BOOL 전자담배점 콜드아웃리치 시작안

## 목적
- 서울 핵심상권 전자담배점의 **스마트콜/대표번호**를 확보
- BOOL 입점 제안을 위한 **콜드 메시지 발송**
- 반응률 높은 매장을 빠르게 선별해 전화/방문 영업으로 연결

---

## 1. 리스트는 이렇게 모은다

단순 전화번호 리스트가 아니라, **영업 우선순위 리스트**로 모은다.

### 필수 항목
- 매장명
- 상권
- 주소
- 스마트콜/대표번호
- 지도 링크
- 인스타/홈페이지
- 리뷰 수 / 평점
- 매장 타입 (전문점 / 멀티샵 / 시가복합 / 무인)
- 매장 톤 (프리미엄 / 대중형 / 할인형 / 트렌디)
- 입지 점수 (1~5)
- 전문성 점수 (1~5)
- 접촉 가능성 점수 (1~5)
- 우선등급 (A/B/C)
- 발송 메시지 타입 (1/2/3)
- 발송 상태 (미발송 / 발송 / 응답 / 후속)
- 메모

### 점수 기준
#### 입지 점수
- 5: 강남역, 홍대, 성수, 잠실 등 핵심상권 메인 동선
- 4: 준핵심상권, 오피스/주거 혼합 밀집
- 3 이하: 외곽 / 유동 낮음

#### 전문성 점수
- 5: 전자담배 전문점 성격 명확
- 4: 멀티샵이지만 전자담배 비중 큼
- 3: 잡화 성격 혼합

#### 접촉 가능성 점수
- 5: 전화번호 + 지도 + 활동 흔적 명확
- 4: 번호 확인 가능, 최근 흔적 있음
- 3: 번호만 있거나 정보 부족

### 우선등급
- **A**: 바로 메시지 + 전화 후보
- **B**: 메시지 우선
- **C**: 보류

---

## 2. 아웃리치 운영 원칙

1. **무차별 대량 발송 금지**
   - 처음엔 A등급 20~30개만 테스트

2. **메시지 짧게**
   - 길게 설명하면 광고처럼 보임

3. **매장 입장에서 바로 이해되게**
   - 제품 특성
   - 마진 포인트
   - 오프라인 판매 적합성
   - 샘플/상담 연결

4. **메시지 후 전화/방문 연결**
   - 문자만 보내고 끝내면 전환이 낮음

---

## 3. BOOL 콜드메시지 문안

### 타입 1 — 기본형
안녕하세요. BOOL KOREA입니다.
전자담배 전문점에서 판매 가능한 니코틴파우치 브랜드 BOOL 소개드리고 싶어 연락드렸습니다.
오프라인 판매 적합성과 매장 마진 구조가 좋아 간단히 안내드리고 싶습니다.
괜찮으시면 제품 소개 자료 보내드려도 될까요?

### 타입 2 — 마진/실무형
안녕하세요. BOOL KOREA입니다.
매장에서 바로 판매 가능한 니코틴파우치 BOOL 제안드리려고 연락드렸습니다.
권장가 기준 매장 마진 구조가 괜찮고, 전자담배 전문점에서 제안드리기 좋은 상품입니다.
원하시면 간단한 소개자료와 조건 먼저 보내드리겠습니다.

### 타입 3 — 짧은 테스트형
안녕하세요. BOOL KOREA입니다.
전자담배 매장용 니코틴파우치 입점 제안드리려고 연락드렸습니다.
부담 없이 보실 수 있는 짧은 소개자료 있는데 보내드려도 괜찮을까요?

---

## 4. 후속 메시지

### 1차 발송 후 무응답 (1~2일 뒤)
안녕하세요. BOOL KOREA입니다.
지난번에 니코틴파우치 BOOL 관련 간단히 연락드렸는데,
혹시 소개자료 받아보실 의향 있으실지 다시 한번 여쭤봅니다.

### 자료 보낸 뒤 후속
안녕하세요. BOOL KOREA입니다.
보내드린 BOOL 자료 확인 가능하셨을까요?
가능하시면 통화나 방문으로 5분 정도만 설명드리고 싶습니다.

---

## 5. 추천 실행 순서

### 1단계
- 강남역 / 신논현 / 역삼 A등급 20개 선정

### 2단계
- 메시지 타입 1, 2, 3 각각 소량 테스트
- 어떤 문안이 응답률 높은지 확인

### 3단계
- 응답 매장은 전화/방문 우선
- 무응답은 1회만 후속

### 4단계
- 반응 좋은 상권부터 확대

---

## 6. claw 의견

처음부터 100개를 한 번에 치는 것보다,
**A등급 20~30개에 문안 테스트를 먼저 돌리는 게 훨씬 효율적**이다.
반응률이 나오는 문구를 찾은 뒤 확장해야 실제 전환이 올라간다.
