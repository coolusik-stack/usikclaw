# Premium Report Template

Supanova의 프리미엄 무드(강한 첫인상, 카드 구조, 한국어 타이포 품질)를 참고하되, 랜딩페이지가 아니라 **경영진/클라이언트용 웹 리포트**에 맞게 재구성한 템플릿입니다.

## 포함 파일

- `report-template.html` — 재사용용 기본 템플릿
- `example-report.html` — 완성 예시 리포트
- 이 템플릿에는 **프린트용 스타일(@media print)** 이 이미 포함되어 있습니다.

## 빠르게 사용하는 방법

1. `report-template.html`을 복사해서 새 파일을 만듭니다.
   - 예: `client-q2-report.html`
2. 아래 플레이스홀더를 실제 내용으로 바꿉니다.
   - `{{REPORT_TITLE}}`
   - `{{REPORT_SUMMARY}}`
   - `{{METRIC_1_VALUE}}` 등 수치 영역
   - `{{EXEC_SUMMARY}}`, `{{HIGHLIGHT_*}}`, `{{ACTION_*}}` 등 본문 영역
3. 브라우저에서 열어 확인합니다.
4. PDF가 필요하면 브라우저의 **인쇄 / PDF 저장**을 사용합니다.

## 어떤 식으로 바꾸면 좋은가

### 1) 히어로 섹션
첫 화면은 “이번 보고서가 왜 중요한지”를 한 문장으로 보여주는 용도입니다.
- 제목: 짧고 강하게
- 요약: 2~3문장
- 핵심 지표: 3개 정도만 배치

### 2) 카드 구조
아래쪽 흰 배경 영역은 카드 단위로 읽히도록 설계되어 있습니다.
- 카드 1장 = 결론 1개
- 긴 설명은 2~4문단보다 **짧은 블록 여러 개**가 더 잘 읽힙니다.
- 경영진용이면 문장보다 숫자 + 해석 조합이 좋습니다.

### 3) 한국어 카피 톤
이 템플릿은 Pretendard와 `word-break: keep-all`을 전제로 잡았습니다.
그래서 문장은 아래처럼 쓰는 쪽이 잘 어울립니다.
- 번역투보다 자연스러운 한국어
- 과장된 표현보다 구체적인 표현
- “혁신적”, “차세대” 같은 빈 표현보다 실제 변화와 근거

## 프린트 / PDF 팁

이미 포함된 프린트 스타일이 다음을 처리합니다.
- 배경을 인쇄 친화적으로 정리
- 장식성 요소 일부 제거
- 카드 내부 페이지 분리 방지
- A4 기준 여백 적용

추가로 추천하는 사용 팁:
- 브라우저 인쇄 옵션에서 **배경 그래픽 포함**을 켜면 더 예쁘게 나옵니다.
- 너무 긴 카드가 생기면 카드 내용을 둘로 나누는 편이 PDF에서 안정적입니다.
- 강제 페이지 분리가 필요하면 `page-break-mobile` 같은 클래스를 응용하면 됩니다.

## 추천 사용 사례

- 분기 성과 보고서
- 브랜드/마케팅 리뷰
- 프로젝트 결과 보고서
- 전략 제안서 초안
- 투자자/임원 공유용 요약 페이지

## 수정 포인트

### 색감 변경
상단 다크 히어로 분위기를 바꾸고 싶으면 아래 값을 먼저 보면 됩니다.
- `tailwind.config` 안의 `colors`
- `body`의 `background` 그라디언트
- `.metric-card`, `.glass`, `.report-card`

### 섹션 추가
기본 구조는 아래 흐름입니다.
1. Hero
2. Executive Overview
3. Highlights
4. Deep Dive / Priority
5. Action Plan

필요하면 이 사이에 아래 섹션을 추가하면 잘 어울립니다.
- 차트 이미지 섹션
- 고객 코멘트 섹션
- 비교표 섹션
- 부록/메모 섹션

## 시작할 때 제일 쉬운 방법

가장 빠른 방법은:
1. `example-report.html`을 먼저 열어 톤을 확인하고
2. 그 파일을 복사한 뒤
3. 텍스트와 수치만 교체하는 방식입니다.

처음부터 구조를 만질 필요가 없다면 이 방법이 제일 빠르고 결과도 안정적입니다.
